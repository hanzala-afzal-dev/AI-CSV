# Domain Glossary

## Dataset

A user-owned CSV object and its lifecycle state.

## Dataset Version

An immutable profile snapshot created after ingestion or re-profiling.

## Analysis Thread

A conversation scoped to one owner and dataset.

## Deterministic Tool

Code that executes a bounded calculation or query and returns verifiable output.

## Retrieval Context

Trusted, filtered knowledge returned from Qdrant or another approved source.
